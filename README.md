* **Node** - v10.x.x (preferrably v10.15.0 for long term support)
* **npm** - v6.x.x (preferrably v6.6.0)

## Running It Locally

Clone this repo via to your local machine and install the dependencies by doing the following:

```bash
git clone 
cd Dvote
npm install
```
